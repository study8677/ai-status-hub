# Support

Use GitHub Issues for support:

- Bugs: use the bug report template.
- Feature requests: use the feature request template.
- New service sources: include the official public status URL and a sample payload when possible.

Before opening an issue, check:

- The live dashboard: https://study8677.github.io/ai-status-hub/
- The latest JSON: https://study8677.github.io/ai-status-hub/last_run.json
- The workflow runs: https://github.com/study8677/ai-status-hub/actions/workflows/monitor.yml

Do not include API keys, credentials, private internal URLs, or personal data in public issues.
