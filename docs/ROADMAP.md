# Roadmap

This project is intentionally small, dependency-free, and official-source first. The roadmap focuses on better signal quality and easier reuse.

## Near-term

- Add per-service filtering for AWS AI-related services such as Bedrock when official data supports it cleanly.
- Add optional webhook output for Slack, Feishu, Discord, and generic HTTP receivers.
- Add service-level configuration for warning/critical overrides.
- Add a compact embeddable badge endpoint generated as static SVG.

## Quality

- Expand parser fixtures for real official payloads.
- Add snapshot tests for the generated dashboard.
- Add accessibility checks for the static page.

## Product

- Add English/Chinese language toggle.
- Add a screenshot and share card for the GitHub README.
- Add a one-command bootstrap script for forks.
