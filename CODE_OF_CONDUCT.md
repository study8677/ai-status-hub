# Code of Conduct

This project follows a simple standard: be respectful, specific, and evidence-driven.

## Expected Behavior

- Use clear technical language.
- Provide official source links and sample payloads when reporting status issues.
- Disagree with claims using evidence, not personal attacks.
- Keep issues and pull requests focused on the project.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Posting secrets, tokens, private URLs, or personal data.
- Repeated low-effort reports without evidence.

## Enforcement

Maintainers may edit, close, or delete issues and comments that violate this standard.
